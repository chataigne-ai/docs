---
title: Customer
---

| Field            | Type                                       | Description                                                                   |
| ---------------- | ------------------------------------------ | ----------------------------------------------------------------------------- |
| `id`             | string                                     | Unique identifier of the user                                                 |
| `firstName`      | string (optional)                          | First name of the user                                                        |
| `lastName`       | string (optional)                          | Last name of the user                                                         |
| `whatsappNumber` | string                                     | Whatsapp Number of the customer                                               |
| `phone`          | string (optional)                          | The phone number of the customer (may be different from the `whatsappNumber`) |
| `address`        | [Address](../common/address.md) (optional) | The last known address of this customer                                       |
| `deliveryNotes`  | string (optional)                          | The last known delivery notes of this customer                                |
