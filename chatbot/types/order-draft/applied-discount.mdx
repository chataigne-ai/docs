---
title: Applied Discount
---

There exist different type of applied discounts:

## Percentage

|                  |                                      |     |
| ---------------- | ------------------------------------ | --- |
| `name`           | string                               |     |
| `discountId`     | string                               |     |
| `priceOff`       | [Price](/chatbot/types/common/price) |     |
| `type`           | "percentage"                         |     |
| `percentage`     | number                               |     |
| `appliedToTotal` | [Price](/chatbot/types/common/price) |     |

## Fixed

|              |                                      |     |
| ------------ | ------------------------------------ | --- |
| `name`       | string                               |     |
| `discountId` | string                               |     |
| `priceOff`   | [Price](/chatbot/types/common/price) |     |
| `type`       | "fixed"                              |     |

## BOGO

|              |                                      |     |
| ------------ | ------------------------------------ | --- |
| `name`       | string                               |     |
| `discountId` | string                               |     |
| `priceOff`   | [Price](/chatbot/types/common/price) |     |
| `type`       | "bogo"                               |     |
| `freeItems`  | [Free Item](#free-item)[]            |     |

## Free Product

|              |                                      |     |
| ------------ | ------------------------------------ | --- |
| `name`       | string                               |     |

| `priceOff`   | [Price](/chatbot/types/common/price) |     |
| `type`       | "free_product"                       |     |
| `freeItem`   | [Free Item](#free-item)              |     |
