|                   |                                            |                                                       |
| ----------------- | ------------------------------------------ | ----------------------------------------------------- |
| `id`              | `String`                                   | id of the delivery                                    |
| `provider`        | `"store"` \| `"uberDirect"` \| `"chaskis"` |                                                       |
| `status`          | [Delivery Status](#delivery-status)        |                                                       |
| `deliveryAddress` | [Address](/chatbot/types/common/address)   |                                                       |
| `etaTime`         | `String`                                   | ISO format in the timezone of the location            |
| `collectionCode`  | `String`                                   | Collection code of the order. Not used for the moment |
| `trackingUrl`     | `String`                                   | URL to track the order                                |
| `fees`            | [Price](/chatbot/types/common/price)       |                                                       |
| courier           | [Courier](#courier)                        |                                                       |

### Delivery Status

- pending
- assigned
- pickup_imminent
- en_route
- dropoff_imminent
- delivered
- cancelled
- failed
