<Info>
  Les services sont la couche qui gère toutes les interactions avec les systèmes
  externes à l'application. Ils encapsulent la logique de communication avec les
  APIs tierces et les services externes.
</Info>

## Rôle des Services

Les services :

- Gèrent les interactions avec les APIs externes
- Encapsulent la logique de communication
- Transforment les données entre notre domaine et les formats externes
- Gèrent les erreurs
