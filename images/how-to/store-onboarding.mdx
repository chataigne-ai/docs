## Initial Setup

<Steps>
  <Step title="Create a new store location">
    Start by creating a new store location in the system.
  </Step>
  <Step title="General Settings">
    Go to `settings/general` to configure: - Payment settings - Delivery
    settings
  </Step>
</Steps>

## Integrations

<Steps>
  <Step title="Hubrise Setup">
    1. Go to `settings/integrations` 2. Complete the Hubrise configuration 3.
    Import the catalog from Hubrise integration settings
    <Note>
      The import might trigger a temporary error modal, but the process works
      correctly.
    </Note>
  </Step>
  <Step title="WhatsApp Setup">
    1. Complete the WhatsApp integration 2. Go to WhatsApp settings 3. Edit the
    catalog you use
  </Step>
</Steps>

## Final Configuration

<Steps>
  <Step title="Flow Setup">
    Create the flow and add the necessary parameters, mainly flow category IDs.
  </Step>
  <Step title="Stripe Integration (Optional)">
    If you use Stripe for payments, complete the Stripe integration process.
  </Step>
</Steps>

<Note>Once all these steps are completed, your chatbot is ready to use!</Note>
