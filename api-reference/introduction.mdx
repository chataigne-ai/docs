## Vue d'ensemble

L'API interne de châtaigne est organisée autour des concepts REST. Elle sert d'interface de communication entre nos différents services :

- Le chatbot WhatsApp
- Le back-office de gestion
- Les intégrations avec les services externes (Hubrise, Stripe, etc.)
- Les webhooks et callbacks

## Authentication

L'authentification des requêtes internes se fait via un token Bearer dans l'en-tête HTTP :

```bash
Authorization: Bearer votre_token_api
```

<Note>
  Les tokens d'API sont générés et gérés automatiquement par notre système
  d'authentification interne.
</Note>
